Centralized Web Analytics

A standalone, zero-dependency GA4 script designed for modern web applications. Scale your tracking without repeating code.

Começar Agora GitHub

Por que centralizar?

Configuração Única

Mantenha a lógica do GA em um só lugar. Altere no script central e reflita em todas as suas aplicações instantaneamente.

Agnóstico a Framework

Funciona nativamente em Next.js, React, Vue, Svelte ou sites estáticos puramente em HTML.

Security First

Validação rigorosa de IDs via regex e sanitização de ambientes para evitar injeções e ataques XSS.

Como Usar

index.html 
<script
  src="https://sua-cdn.com/analytics.js"
  data-ga-id="G-XXXXXXXXXX"
  data-env="prod"
  data-debug="false"
  async>
</script>
layout.tsx 
import Script from 'next/script'

export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        {children}
        <Script
          src="https://sua-cdn.com/analytics.js"
          data-ga-id={process.env.NEXT_PUBLIC_GA_ID}
          strategy="afterInteractive"
        />
      </body>
    </html>
  )
}
index.html 
<!-- No arquivo principal de entry do Vite -->
<script
  src="/analytics.js"
  data-ga-id="%VITE_GA_ID%"
  data-env="staging"
  data-debug="true"
  async
></script>

API de Tracking

Utilize a função global window.track para eventos customizados de forma segura.

javascript 
// Clique em um botão de compra
window.track('purchase_click', {
  item_id: 'prod_123',
  value: 49.90,
  currency: 'BRL'
});

Segurança e CSP

Se você utiliza CSP, adicione os seguintes domínios à sua política:

Content-Security-Policy 
script-src 'self' https://www.googletagmanager.com;
connect-src https://www.google-analytics.com https://stats.g.doubleclick.net;
